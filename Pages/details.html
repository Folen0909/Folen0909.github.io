Fragment User

Fragment
ProjectLOGS

This is the first thing you see when you open the app. Here you can see the details of the user who is using the app. Informations you can get are: image, name, location and short description.
This fragment is part of view pager along side with Build fragment. Fragment contains one button that enables you to edit user info.

Fragment Build

Fragment
ProjectLOGS

This view contains recycler view which contains all of the builds of that user. It is possible to add new builds by pressing on the button.
Builds can be deleted on single press, this will delete all data in that build. After pressing on any build new fragment is opened, component fragment.

Fragment Component

Fragment

Component fragment is the last fragment at this time. It is used for showing all components in single build.
It is using recycler view in which items are represented by cards. Each card contains the components name and image.
